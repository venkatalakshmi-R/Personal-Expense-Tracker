# Personal Expense Tracker

## Description

This project was developed using Microsoft Excel to track and analyze daily personal expenses. The system records expenses, categorizes spending, and generates summaries for better financial management.

## Features

* Daily Expense Recording
* Category-wise Expense Tracking
* Total Expense Calculation
* Spending Analysis using Excel Formulas

## Categories Used

* Food
* Travel
* Shopping
* Recharge

## Formulas Used

* SUM()
* SUMIF()

## Tools Used

* Microsoft Excel 2016

## Outcome

The project helps users monitor spending habits and manage personal finances effectively.
